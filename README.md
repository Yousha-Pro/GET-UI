# 🚀 GET-UI (TypeUI Design Extractor)

A powerful Chrome extension that extracts **real website UI systems** and converts them into structured, AI-ready design documentation.

It analyzes any webpage and generates:

- `DESIGN.md` → full design system blueprint  
- `SKILL.md` → component behavior + interaction logic  
- `PROMPT.md` → AI-ready UI reconstruction prompt  

Built for developers, designers, and AI-assisted workflows.
