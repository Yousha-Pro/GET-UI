> **`PROMPT.md` — how it actually works:** Inside the extension, select any part of a website — a single button, a card, a hero, a navbar, or multiple sections at once. GET-UI reads your exact selection and writes a precision AI prompt for it. Paste it into Claude, Cursor, or Codex and get that exact piece rebuilt in your stack, your language, your framework. No prompt writing. No guesswork. Pixel-targeted output.

## ⚙️ &nbsp; What Gets Extracted

<br/>

<details>
<summary><b>🎨 &nbsp; Design Tokens — click to expand</b></summary>

<br/>

```
Colors
  ├── Backgrounds       surface · page · overlay · elevated
  ├── Text              primary · secondary · muted · inverted · link
  ├── Borders           default · strong · subtle · focus
  ├── Brand             primary · secondary · accent
  └── Semantic          success · warning · error · info

Typography
  ├── Scale             display · h1–h4 · body · caption · label
  ├── Weights           all values in use across the site
  ├── Line heights       per level
  ├── Letter spacing     per level
  └── Font families      primary · mono · display

Spacing
  ├── Base unit          auto-detected (4px, 8px, 5px, etc.)
  ├── Scale              full spacing token set
  ├── Padding patterns   common component padding
  └── Margin rhythm      section and layout spacing

Effects
  ├── Shadows            named levels + raw values
  ├── Border radius      component-level tokens
  ├── Blur               backdrop + drop values
  └── Motion             easing curves + durations
```

</details>

<details>
<summary><b>🧩 &nbsp; Components — click to expand</b></summary>

<br/>

```
UI Primitives
  ├── Buttons            variants · sizes · states · icons
  ├── Inputs             text · select · checkbox · radio · toggle
  ├── Badges             colors · shapes · with/without icons
  └── Avatars            sizes · fallback · group

Composition
  ├── Cards              layout · header/body/footer · media
  ├── Modals             overlay · sizing · close behavior
  ├── Navigation         top bar · sidebar · mobile collapse
  ├── Tabs               indicator · alignment · scrollable
  └── Accordions         animation · multiple open

Layout
  ├── Grid               columns · gap · breakpoints
  ├── Flex               direction · alignment · wrap
  ├── Section groups     vertical rhythm · max-width · centering
  └── Visual hierarchy   z-index · stacking · elevation

Behavior
  ├── Hover states        color shift · underline · shadow
  ├── Focus states        outline · ring · border
  ├── Active/Pressed      scale · brightness
  ├── Disabled            opacity · cursor · interaction lock
  └── Loading             skeleton · spinner · placeholder
```

</details>
